O Git é sistema que permite o controle de versões no desenvolvimento de um software, a partir dele é possível ver as mudanças no código fonte dos arquivos, também permite um controle melhor quando um grupo de desenvolvedores trabalham em um mesmo arquivo permitindo que seja possível juntar o trabalho de todos verificando as mudanças que ocorreriam se isso acontecesse.

Comandos Utilizados (Todos os comandos foram feitos utilizando o terminal, Prompt de Comando e Git Bash):

1º gh repo create		// criar o repositorio
2º git pull origin main 	// da pull na branch
3º git checkout main	// muda para branch main
4º git add .		// adicionar os arquivos
5º git commit  -m "Commit Inicial" 		// commit dos arquivos
6º git push origin main	// para mandar os arquivos diretos para o repositorio
